## üèó About the Project

This project was a freelance commission to build a digital presence for **United Mode Trading**. The client required a fast, SEO-friendly, and mobile-responsive website to showcase their portfolio and product offerings to the Saudi market.

The website highlights the company's role in the construction sector, specifically focusing on:
* Supply of Electrical & Plumbing materials.
* Construction trading services.
* Company portfolio and location details (Riyadh).

---

## ‚ú® Key Features

* **Responsive Design:** Fully adaptive UI built with **Tailwind CSS** that looks great on mobile, tablet, and desktop.
* **Client-Side Routing:** Seamless navigation between pages (Home, Products, About, Contact) using **React Router DOM**.
* **Product Showcase:** dedicated sections for displaying construction materials (Electrical, Plumbing, etc.).
* **Interactive UI:** Modern components and layout for a professional corporate look.
* **Performance:** Optimized assets and fast loading times typical of a React SPA.
